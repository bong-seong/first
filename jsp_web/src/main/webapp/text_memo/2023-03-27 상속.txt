상속	
	목적  : 재활용성
	클래스A extends 클래스B
	1. 자식클래스가 부모클래스 선택
	2. 하나의 부모클래스에게 상속을 받는다.
	3. 자식객체 ---> 부모객체
		Object O = [ String ] "유재석"
		부모객체 ---> 자식객체

인터페이스
	목적 : 동일한 목적하의 서로 다른 객체[클래스]의 기능[함수] 수행 하게끔
	클래스A implements 인터페이스A , 인터페이스 B
	1. 인터페이스 추상메소드 [ 선언 O 정의 X ] 존재 / 변수X 상수O
	2. 여러개의 인터페이스를 구현할 수 있다.
	* 다양한 API 사용할때 해당 메소드/필드 타입 확인
		인터페이스 구현객체
